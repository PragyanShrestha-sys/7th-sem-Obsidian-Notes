## Financial, Technical, and Time-Based Cost Monitoring Tools

---
### General Definition

Cost monitoring tools are instruments used to **track, measure, and report** project expenditures across three dimensions:

| Cost Type | Definition |
|-----------|------------|
| **Financial** | Tools that track monetary spending against budget |
| **Technical** | Tools that monitor resource consumption (cloud, infrastructure, etc.) |
| **Time-Based** | Tools that track labor hours, effort, and schedule adherence |

---

### Tools by Category

| Category | Definition | Examples |
|----------|------------|----------|
| **Financial** | Track actual money spent vs. budget | Excel, QuickBooks, SAP, MS Project |
| **Technical** | Monitor computing resource usage | AWS Cost Explorer, Datadog, Kubecost |
| **Time-Based** | Track effort and schedule performance | Toggl, Gantt charts, Slip Charts, Jira burndown |

---

### Key Metrics

| Category       | Key Metrics                               |
| -------------- | ----------------------------------------- |
| **Financial**  | Actual Cost, Budget Variance, CPI         |
| **Technical**  | CPU, Storage, API calls, Data transfer    |
| **Time-Based** | Actual Hours, Schedule Variance, Slippage |
